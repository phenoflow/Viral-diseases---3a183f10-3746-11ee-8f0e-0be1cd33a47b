# Viral-diseases

Viral diseases (excl chronic hepatitis/HIV) - PH335

## Requirements

[Docker](https://docs.docker.com/install/) and [CWLTool](https://github.com/common-workflow-language/cwltool#install)

## Configuration

Specify input dataset path in [Viral-diseases-inputs.yml](Viral-diseases-inputs.yml).

## Usage

Run: `cwltool Viral-diseases.cwl Viral-diseases-inputs.yml`
